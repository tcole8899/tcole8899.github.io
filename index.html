UM Demo Instructions

  1. Download the demo
  2. Extract contents of the zip archive
  3. Open a linux terminal
  4. Navigate to the ExecutableUM directory
  5. run "./um ./Programs/advent.umz" (a text-adventure game)
  6. run "./um ./Programs/codex.umz" (a 32-bit OS with a puzzle)

To compile a personal version clone the repository
and follow the instructions in the provided README
